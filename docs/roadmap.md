# MVP e roadmap

## MVP (8 a 12 semanas)
- autenticação local + verificação de email
- homepage, exploração, detalhe do evento, grupos
- RSVPs, lista de espera, bilhetes, checkout
- planos Gratuito / Silver / Gold
- dashboards base
- admin essencial
- Stripe + gateway proprietário

## Fase 2
- PayPal completo
- login social
- app mobile híbrida
- chat em tempo real
- notificações push nativas
- antifraude mais robusto
- recomendação com scoring mais avançado

## Fase 3
- eventos recorrentes avançados
- patrocínios self-service
- marketplace de parceiros
- programa de afiliados
- IA para recomendações, resumo de discussões e moderação assistida
